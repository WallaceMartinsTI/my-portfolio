> 🇧🇷 [Português](#português) · 🇺🇸 [English](#english)

## Português
 
Portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro, hospedado via GitHub Pages.
 
### 📋 Sobre
 
Site de currículo e portfólio com seções de apresentação, experiência profissional, habilidades técnicas, projetos em destaque e contato.
 
### ✨ Funcionalidades
 
- Carrossel de projetos com autoplay, navegação por botões e suporte a swipe
- Animações de entrada com Intersection Observer
- Layout responsivo para mobile e desktop
- Tema escuro com tipografia e paleta de cores customizadas
 
### 🛠️ Tecnologias
 
- HTML5
- CSS3 (variáveis, grid, flexbox, animações)
- JavaScript puro (sem frameworks)

## English
 
Personal portfolio built with plain HTML, CSS and JavaScript, hosted on GitHub Pages.
 
### 📋 About
 
A resume and portfolio website featuring sections for introduction, professional experience, technical skills, highlighted projects and contact info.
 
### ✨ Features
 
- Project carousel with autoplay, button navigation and swipe support
- Scroll-triggered fade-in animations via Intersection Observer
- Responsive layout for mobile and desktop
- Dark theme with custom typography and color palette
 
### 🛠️ Tech Stack
 
- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (no frameworks)
 
### 🚀 Running locally
 
```bash
git clone https://github.com/WallaceMartinsTI/my-portfolio.git
cd my-portfolio
```
 
Open `index.html` directly in your browser, or use an extension like **Live Server** on VS Code.
